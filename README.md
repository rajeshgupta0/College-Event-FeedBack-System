# Pragyan – AI-Powered College Event Feedback & Intelligence Platform

Pragyan is a modern AI-powered platform designed to help educational institutions collect, analyze, and transform student feedback into actionable insights. The system combines intelligent sentiment analysis, event analytics, recommendation engines, and administrative dashboards to improve student engagement and event quality.

## Overview

Traditional feedback systems often collect large amounts of student feedback but fail to generate meaningful insights. Pragyan addresses this challenge by leveraging Artificial Intelligence to automatically analyze feedback, identify trends, detect urgent issues, and provide data-driven recommendations.

The platform serves both students and administrators:

* Students can submit event feedback, ratings, and reviews.
* Administrators gain access to advanced analytics and AI-generated insights.
* Institutions can make informed decisions based on real student experiences.

## Installation

### Clone Repository

```bash
git clone https://github.com/rajeshgupta0/College-Event-FeedBack-System.git
```

### Navigate to Project

```bash
cd College-Event-FeedBack-System
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_publishable_key
```

### Run Development Server

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
