# Pragyan - AI-Powered College Event Feedback System

## About Pragyan

**URL**: [Pragyan Platform](https://pragyan.app)

## How to Edit This Code

You have several ways to customize and enhance Pragyan:

### Option 1: Use Lovable (No Setup Required)

Simply visit the [Pragyan Project Dashboard](https://lovable.dev/projects/c4752987-5068-48f6-95f8-ade5b899fe05) and start giving instructions.

Any changes you make through Lovable will be automatically saved and committed to this repository.

### Option 2: Use Your Preferred Code Editor

If you want to work locally on your machine, you can clone this repository and make changes. Your updates will sync back to Lovable.

**Requirements:**
- Node.js installed (download from [nodejs.org](https://nodejs.org/))
- npm (comes with Node.js)

**Setup Steps:**

```bash
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project folder
cd pragyan-event-feedback-system

# Step 3: Install all dependencies
npm install

# Step 4: Start the development server
npm run dev
Technology Stack

Pragyan is built with modern, production-ready technologies:

    Frontend Framework: React 18 with TypeScript

    Build Tool: Vite for fast development and optimized builds

    Styling: Tailwind CSS with shadcn-ui components

    Backend & Database: Supabase (PostgreSQL + Auth + Storage)

    AI Integration: Google Gemini AI for sentiment analysis

    Charts & Analytics: Recharts for beautiful data visualization

    Animations: Framer Motion / CSS Animations

    Edge Functions: Deno for serverless AI processing
